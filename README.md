
# movie_info_app
Objective-To create web app for finding Movie details
Technology Used-Javascript,Bootstrap,Css,Jquery,Html,OMDB_API;

•Landing page(home.html) :- This page will be the ﬁrst page a user visits when they visit this web app. It should tell the description about the web app like what this website is all about. It should also contain information on diﬀerent genres available on the website and navigation option to the each genres page.
•Genres page(index.html) :- This page should contain list of movies in the genre and on clicking any movie it should open a movie page.
•Movie page(movie.html) :- This page should contain details about movie selected. Details should contain description of movie, ratings, release date etc.
